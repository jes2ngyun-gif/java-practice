## 합격자 이름만 ArrayList에 저장하기
> 배열은 처음부터 값들이 들어있는 원본 데이터
> 
> ArrayList는 조건에 맞는 값만 따로 담는 결과 바구니

---
### 문제 설명
학생 이름 배열과 점수 배열이 있다.

점수가 80점 이상인 학생 이름만 ArrayList에 저장하고 출력한다.

---
### 다루는 문법

| 문법        | 사용 위치             |
| --------- | ----------------- |
| 배열        | 이름, 점수 원본 저장      |
| ArrayList | 합격자 이름 저장         |
| 반복문       | 학생 전체 확인          |
| 조건문       | 80점 이상인지 판단       |
| 메서드       | 합격자 찾기 기능 분리      |
| 제네릭       | ArrayList<String> |

---
### 원리 그림
```
names  = ["정윤", "철수", "영희", "민수", "수진"]
scores = [  90,    70,    85,    60,    100 ]

반복문으로 같은 인덱스를 확인한다.

i = 0 → 정윤 / 90점  → 합격 → passedNames에 추가
i = 1 → 철수 / 70점  → 불합격 → 추가 안 함
i = 2 → 영희 / 85점  → 합격 → passedNames에 추가
i = 3 → 민수 / 60점  → 불합격 → 추가 안 함
i = 4 → 수진 / 100점 → 합격 → passedNames에 추가
```
결과:
```
passedNames = ["정윤", "영희", "수진"]
```
---
### 실행 흐름도
```
프로그램 시작
   ↓
이름 배열 생성
   ↓
점수 배열 생성
   ↓
빈 ArrayList 생성
   ↓
반복문으로 점수 확인
   ↓
80점 이상이면 이름을 ArrayList에 추가
   ↓
합격자 목록 출력
   ↓
프로그램 종료
```
---
### 코드 작성하기 전에 생각하는 순서
1. 학생 이름 여러 개가 필요하다. → String[] names
2. 점수 여러 개가 필요하다. → int[] scores
3. 합격자 이름만 `따로 담아`야 한다. → `ArrayList<String> passedNames`
4. 이름과 점수는 같은 인덱스로 연결된다.
5. 점수가 80점 이상이면 passedNames.add(names[i])
