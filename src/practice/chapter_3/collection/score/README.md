### 문제 제목 : 학생 점수 관리 시스템
HasgMap을 익히기 위한 예제. HashMap 실무에서 자주 사용됌.

### 문제 설명
학생 이름과 점수를 저장하는 프로그램을 만든다.
- 학생 등록
- 학생 점수 조회
- 특정 학생 존재 여부 확인
- 학생 삭제
- 전체 학생 출력

를 수행한다.

### 왜 HashMap이 필요한가?
학교 출석부에는
```
1번 김민수
2번 박철수
3번 이영희
```
처럼 번호가 붙어있다.

은행에서도
```
계좌번호 → 돈
```
이 연결되어 있다.

즉,
```
이름표(key)
↓
실제 값(value)
```
형태로 관리하는 경우가 많다.

HashMap은 이런 문제를 다루기 위해 사용된다.

---
### 원리
```
학생 이름(key)
      ↓
+-------------------+
| 김민수 → 95점       |
| 박철수 → 80점       |
| 이영희 → 100점      |
+-------------------+
      ↓
HashMap
```
### 실행 흐름도
```
main 시작
    ↓
    
HashMap 생성
    
    ↓
학생 등록(put)
    
    ↓
특정 학생 점수 조회(get)
    
    ↓
학생 존재 여부 확인(containsKey)
    
    ↓
학생 삭제(remove)
    
    ↓
전체 학생 출력(for-each)
    
    ↓
프로그램 종료
```

### 반드시 알고 넘어가야 할 것.
## 1.
```
HashMap<Key, Value>
```
는
```
이름표(key)
↓
실제 값(value)
```
을 연결해서 사용한다.

## 2.
```
scoreMap.get("김민수");
```
하면:
```
95
```
가 나온다.
## 3.
실무에서는 get( )을 하기 전에
```
containsKey()
```
로 존재 여부를 먼저 확인하는 경우가 많다.
